O que é HTML?

HTML é uma linguagem de marcação desenvolvida por Tim Berners-Lee com a finalidade de criar documentos de hipertexto que se comuniquem entre si.

Como funciona?

Diferente das linguagens de programação, a linguagem de marcação HTML funciona por meio de tags;

O que são as CSS?

As CSS utilizam de seletores, propriedades e valores para atribuir estilo a um documento HTML.

HTML e CSS são linguagens de programação?

Não! Uma linguagem de programação depende do uso de instruções, variáveis, condições, laços e até objetos. Mas nada disso existe em HTML ou CSS.

No entanto, é possível fazer uso de linguagens de programação como PHP e Javascript para criar interações com seu site, por exemplo!

Como funcionam as tags

Tags em HTML sempre ficam dentro de colchetes angulares

Exemplo: <p> Isso é um parágrafo em HTML </p>

Percebe-se que a tag de fechamento possuí uma barra (/). Nem todas as tags possuem fechamento, a exemplo temos a tag <img>.

Além disso, as tags podem ter atributos e valores.


Exemplo: <a href="index.html"> Exemplo de Link em HTML </a>

Nesse link, href é um parâmetro, enquanto "index.html" é um valor.

Estrutura de um documento HTML

<!DOCTYPE html>
Expressa que o documento será escrito na versão mais atual da HTML

<html lang="pt-br"> </html>
Delimita o documento HTML e indica que ele será escrito em Português do Brasil

<head> </head>
Delimita o head da página. Nessa sessão serão realizadas configurações de estilos, formato, ícones dentre outros.

<meta charset="UTF-8">
Indica que a página suporta caracteres acentuados.

<meta name="viewport" content="width=device-width,initial-scale=1.0">
Indica que todo o conteúdo aparecerá ocupando todo o espaço da tela com uma escala de 1:1

<title>Hello World!</title>
Texto que aparecerá na aba do navegador, ao lado do favicon.

<body> </body>
Corpo da página. É aqui que grande parte do seu documento será escrito. É onde o conteúdo será exibido.
